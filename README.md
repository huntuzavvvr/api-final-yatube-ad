# API Final Yatube

API Final Yatube — учебный REST API для социальной сети Yatube.  
Проект позволяет работать с публикациями, комментариями, группами и подписками через HTTP-запросы.

## Возможности

- Получение списка публикаций.
- Создание, редактирование и удаление собственных публикаций.
- Получение групп.
- Создание и просмотр комментариев к публикациям.
- Подписка на других пользователей.
- JWT-аутентификация.

## Технологии

- Python
- Django
- Django REST Framework
- Simple JWT
- SQLite
